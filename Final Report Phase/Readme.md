# Project Report: Laptop Request Catalog Item

This folder contains the final university project report for the "Laptop Request Catalog Item" project, completed for the Naan Mudhalvan program.

This document provides a comprehensive, step-by-step explanation of the project's objectives, design, implementation, and results.

## Inside This Report

The full PDF report contains the following sections:

* **1. Abstract & Problem Statement:** A summary of the project and the manual processes it was designed to solve.
* **2. Tools and Platform:** A list of the ServiceNow modules and technologies used, such as Service Catalog, UI Policies, UI Actions, and Update Sets.
* **3. Step-by-Step Explanation:** A detailed walkthrough of the project build, including:
    * Creating the Update Set
    * Building the Service Catalog Item
    * Adding Variables (e.g., `Laptop Type`, `RAM`, `Storage`)
    * Applying UI Policies for dynamic form behavior
    * Adding a "Reset Form" UI Action
* **4. Testing and Deployment:** An overview of how the project was exported and then imported into a target instance for testing.
* **5. Final Output & Conclusion:** Includes a screenshot of the final "Order Status" page and a summary of the project's success.
